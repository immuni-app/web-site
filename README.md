# Immuni.italia.it
A website for the Immuni App.

## Getting started
- Open your terminal
- Clone this repo
- Move to the folder
- Use _npm_ or _yarn_ _install_ to setup the packages (ex. _yarn install_)
- Use _npm_ / _yarn_ _start_ to serve the web site
- Open your browser and visit _http://localhost:8080_
